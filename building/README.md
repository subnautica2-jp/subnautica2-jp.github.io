## 概要

Subnautica 2 では **基地建設システムが刷新** されることが公式に発表されています。前作Subnautica / Below Zero のHabitat Builderベースのシステムから、より柔軟・直感的なシステムへの変更が予告されています。

## 公開済みの方針（Dev Vlog より）

- 基地建設システムは前作から **刷新**
- 詳細はDev Vlog（特に #2〜#4 周辺）で段階的に公開
- マルチプレイでの同時作業を前提とした設計

## 想定される変更点

正確な仕様はEA開始後に確定しますが、刷新方針として以下が想定されます:

- より自由な配置・接続
- マルチプレイで複数プレイヤーが同時建築できる仕組み
- 視覚的な配置プレビューの強化
- 既存パーツの解体・再配置のしやすさ向上

## 確定情報待ちの項目

- 建材の種類・必要素材
- 構造強度・圧力耐性のシステム
- 部屋・通路・モジュールのリスト
- 電源・酸素供給・水・食料生産モジュール
- マルチプレイ時のオーナーシップ・許可システム
- 旧Subnauticaシリーズのプレイヤーに対する変更点詳細
