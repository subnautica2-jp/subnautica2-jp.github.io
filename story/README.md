## ストーリー構成について

公式の発表によると、Subnautica 2 はEarly Accessから開始され、Early Access期間中に段階的にストーリーコンテンツが追加されていくと見られます（前作Subnauticaの開発体制と同様）。

EA開始時点で実装されるストーリー範囲・主要NPC・エンディング条件などは2026年4月時点で公式から詳細公表されていません。
