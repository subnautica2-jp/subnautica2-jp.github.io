# 乗り物概要

<div class="pre-release-notice">
※ 2026年4月時点で公開されている乗り物は Tadpole のみです。EA開始時に追加乗り物が公開される可能性があります。
</div>

Subnautica 2 で公開済みの乗り物。前作Subnauticaでは Seamoth・PRAWN・Cyclops 等の階層的な乗り物が登場しましたが、本作では新規ラインナップとなります（前作の乗り物は登場しない方針）。

## 公開済み乗り物

| 名称 | 種別 | 詳細ページ |
|------|------|-----------|
| **Tadpole** | 潜水艇 | [Tadpole](tadpole.md) |

## 確定情報待ちの項目

- Tadpoleより上位の中型〜大型潜水艇の有無
- 陸上活動用ビークル（Below ZeroのSnowfox相当）の有無
- 乗り物のアップグレードシステム
- マルチプレイ時の同乗仕様
